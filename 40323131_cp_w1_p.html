上課內容

1.建立github倉儲，倉儲名稱不限，新增時Remade打勾 2.git clone 倉儲的網址，下載完成後，輸入"cd 倉儲名稱"，進入倉儲資料夾 3.輸入git branch 確認目前倉儲的所有分支 4.輸入git branch gh-pages 建立名為gh-pages的分支 5.再次輸入git branch 確認gh-pages分支是否有建立 6.輸入git checkout gh-pages 將目前分支切換為gh-pages分支

完成後將reaval的gh-pages裡的檔案download，將除了Readme和.git之外其他檔案複製到自己的資料夾 將檔案推到遠端 1.git add -A 新增檔案 2.git config --global user.name "使用者github名稱" 3.git config --global user.email "使用者github信箱" 4.git commit -m "1001" 為這次版本變動註解 5.git push origin gh-pages 將新增檔案推到gh-pages裡頭 6.認證使用者github帳號及密碼 7.確認是否成功,輸入網址即可
